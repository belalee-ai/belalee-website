## Design Context

### Users
- **HR/招聘方**：浏览简历页，快速了解工作经历和核心成果，通常在工作时间用电脑访问
- **同行/产品经理圈**：看博客洞察和项目经历，寻找可交流的话题
- **潜在合作方**：评估专业能力和经验匹配度，了解背景
- **门面展示**：作为个人品牌的线上名片，给所有来访者留下专业印象

访问场景：白天工作时间为主，PC 端优先但需兼顾手机端分享链接

### Brand Personality
- **声音**：专业但不冷漠，温暖但不随意
- **3 词个性**：专业、温暖、自信
- **情绪目标**：让访客觉得"这个人既有实力又好相处"

### Aesthetic Direction
- **视觉调性**：Airbnb 风格 — 白色画布、单一品牌强调色（Rausch Red #ff385c）、图片优先、慷慨的留白
- **参考**：Airbnb 的温暖克制感、旅行杂志般的浏览节奏
- **反面参考**：不要科技风、不要暗黑模式、不要黑乎乎的配色、不要赛博朋克/霓虹风格
- **主题**：纯白光亮模式，不提供暗色主题
- **字体**：LXGW WenKai（标题/装饰，温暖文艺）+ Noto Sans SC（正文，清晰可读）+ Plus Jakarta Sans（西文，圆润几何）
- **配色**：Rausch Red (#ff385c) 作为唯一品牌强调色，仅用于 CTA 和品牌时刻；#222222 温暖近黑文字；#ffffff 纯白画布
- **阴影**：三层渐进阴影系统（border ring + soft blur + lift）
- **圆角**：8px 按钮、14px 徽章、20px 卡片、32px 大容器、50% 控件

### Design Principles
1. **Photography-first** — 内容和图片是主角，设计退后让内容说话
2. **Single-accent restraint** — Rausch Red 是唯一的强调色，因为稀缺所以有力
3. **Warm professionalism** — 用 #222222（非纯黑）、圆润字体、慷慨圆角传递温暖感，同时保持专业
4. **Magazine-pace whitespace** — 慷慨的留白创造从容的浏览节奏，而非信息堆砌
5. **Tactile elevation** — 三层阴影系统让界面有真实的物理感，而非扁平的 CSS 效果
